# **App Name**: INSD

## Core Features:

- Dynamic Landing Page Content Display: Renders all specified sections (Hero, Value, What You Will Learn, etc.) with provided textual content, ensuring correct branding elements and structural layout guidance.
- Lead Capture Form Integration: Implements the lead generation form with fields for name, phone, email, and course interest, designed for submission to an external lead management system.
- Conversion-Focused Call to Actions: Displays various primary, secondary, and outline CTA buttons (e.g., 'Apply Now', 'Book Free Counselling', 'Get Course Details') in their designated colors and positions.
- Responsive Design System: Ensures the landing page layout and content adapt seamlessly across all devices, from desktop to mobile, with a mobile-first approach and a sticky CTA button on mobile.
- Structured Information Display: Presents course modules using a grid with relevant icons and 'Why Choose INSD' as card-style layouts to improve readability and visual appeal.
- Urgency and Trust Element Integration: Incorporates microcopy and elements like 'Limited Seats Available', 'Next batch starting soon', and optional testimonial placeholders to enhance visitor engagement and conversion.

## Style Guidelines:

- Primary Color: #134a84 (Deep Blue) for headings, navigation bar, and key informational sections, establishing a professional and trustworthy brand presence.
- Accent Color: #db3436 (Red) for prominent call-to-action buttons, highlights, and important accents, ensuring high visibility and driving conversion.
- Background Color: #ffffff (White) provides a clean, modern canvas for maximum readability and visual contrast, aligning with the minimal design aesthetic.
- Headline Font: 'Belleza' (humanist sans-serif) for its elegant, art- and design-aligned personality, suitable for bold, aspirational statements. Body Font: 'Inter' (grotesque sans-serif) for its modern, objective, and highly legible quality, ensuring clear communication of detailed content.
- Utilize clean, modern, and simple line-art or filled icons for content grids (e.g., 'What You Will Learn' section) to visually represent concepts without clutter, enhancing comprehension and engagement.
- Adopt a clean, well-structured layout with clear section spacing and a hierarchical flow. Ensure a mobile-first approach for all elements and implement a sticky CTA button for optimal user experience on smaller screens.
- Incorporate subtle animations for CTA button hover states to encourage interaction, and soft, understated transitions for content elements to create a refined and engaging user experience without being distracting.
